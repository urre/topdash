Like OS X admin bar menu icons, but for WordPress. Easy access to common wp-admin menus in the top right admin bar.

== Description ==

1. Adds icons in the top right admin bar, easy access to common wp-admin menus

* [Github](https://github.com/urre/topdash)

== Installation ==

1. Upload the `topdash` folder to your `/wp-content/plugins/` directory
2. Activate the "Topdash" plugin in your WordPress administration interface
3. Done!
